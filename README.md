c#微信机器人 PC hook 稳定不掉线 微信个人号接口 （自动通过、拉群，踢人，发图片，自动回复）<br><br>

目前可实现功能：<br>
1、根据微信号、wxid、昵称发送文本消息。<br>
2、根据wxid发送图片消息。<br>
3、根据wxid发送卡片链接。<br>
4、自动收取转账。<br>
5、发送群邀请链接<br>
6、静默拉群（小群，拉人对方无消息提醒）<br>
7、群活群成员列表<br>
8、群内发送消息<br>
9、群内@人<br>
10、提群内成员<br>
11、自动通过好友<br>
12、发送视频mp4<br>
13、发送文件
...........
